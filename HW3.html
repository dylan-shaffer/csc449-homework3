  1. Write a version of the Ant Clustering algorithm:
    1. Create a grid of 200 x 200. 
    2. Randomly place 200 objects.  100 red and 100 blue.  
    3. Randomly place 500 ants on the grid.  Ants are unloaded.
    4. Repeat:
      1. For each object
        1. Find the number of same colored objects in the adjacent cell neighborhood:  local density.
        2. Find discrete rules for pickup (unloaded ants) and drop-off (loaded ants) based on the local density.
        3. Move ants one step randomly to an adjacent grid.
        4. Apply pickup and drop-off.  
    5. Plot the result.  Your goal is to find rules that cause the red and blue objects to be clustered together.  
  2. Write the text's Particle Swarm Algorithm, alg 5.4, on the text example 3.3.3 in Python.  Compare to your simulated annealing result.
